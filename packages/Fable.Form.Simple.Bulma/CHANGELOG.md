### Added

* Add support for more field type:
    - `Color`
    - `Date`
    - `DateTimeLocal`
    - `Email`
    - `Number`
    - `Search`
    - `Tel`
    - `Time`

* Allow to pass any attributes to an input field using the new `HtmlAttributes` property
